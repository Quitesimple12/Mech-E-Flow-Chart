# How To Run Mech-E-Flowchart
Open a new codespace
type in the terminal "cd me-flowchart"
then type in the command "npm install"
Wait around 30 seconds
Type in the terminal "npm start"
This will prompt you to open a tab, press "yes".
If you fail to follow prompt, go to "Ports" tab at the bottom, and click on the forwarded address.
